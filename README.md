# MovieRecommenderApp
the dataset used is available at https://www.kaggle.com/datasets/asaniczka/tmdb-movies-dataset-2023-930k-movies  
1.first the data_processing script is used to import the dataset,clean it and embed all the necessary informations in 2 binary files available at https://drive.google.com/drive/folders/1MkBs0lvMNgqbWP7y4aY_cwWx2hNd1hzN  
2.the backend script is the recommendation part,it receives the user input from the frontend(usually keywords) and it does the magic
